# ПОДАЧА Водитель — гибридное Android-приложение

## Что это
Гибридное приложение = твой сайт (driver/index.html) + Android-обёртка.  
Работает в фоне, даже когда телефон спит. Передаёт координаты раз в 10 секунд.

## Как собрать APK (установочный файл)

### Шаг 1: Скачай Android Studio
Ссылка: https://developer.android.com/studio  
Нажми "Download Android Studio", установи. При установке НЕ меняй галочки — всё по умолчанию.  
Это займёт ~15 минут.

### Шаг 2: Открой проект
- Запусти Android Studio
- Нажми "Open" → выбери папку `driver-app/android`
- Дождись загрузки (Gradle скачает всё сам, ~5-10 минут)

### Шаг 3: Собери APK
- В меню: Build → Build Bundle(s) / APK(s) → Build APK(s)
- После сборки появится уведомление "APK(s) generated successfully"
- Нажми "locate" → откроется папка с готовым файлом `app-debug.apk`

### Шаг 4: Установи на телефон
- Скопируй .apk файл на телефон (через USB, Telegram, облако)
- Открой файл на телефоне → "Установить из неизвестных источников" → Разрешить
- Готово! Иконка "ПОДАЧА Водитель" появится на рабочем столе

## Как это работает
1. При запуске открывается WebView с твоим сайтом (driver/index.html)
2. Сразу запускается ForegroundService — в шторке появляется "Водитель онлайн"
3. Сервис каждые 10 секунд шлёт GPS-координаты на сервер
4. Когда приходит заказ — сайт в WebView издаёт звук, сервис продолжает работать
5. Даже если телефон спит — координаты шлются, водитель остаётся онлайн

## Файлы проекта
- `MainActivity.java` — главный экран с WebView
- `ForegroundService.java` — фоновый сервис (GPS + отправка на сервер)
- `AndroidManifest.xml` — разрешения и настройки
- `heartbeat.php` (в проекте на сервере) — принимает координаты без CSRF
